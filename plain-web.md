**Coming Soon / Maintenance Page Prompt**

> Create a single self-contained `index.html` coming soon / maintenance / live redirect page.
>
> **Config** — three boolean toggles at the top of the file:
> - `COMING_SOON` — countdown + notify form
> - `MAINTENANCE` — downtime message with estimated return time
> - `LIVE` — redirects to `LIVE_URL`
> Only one should be `true` at a time.
>
> **Coming Soon page**
> - Large elegant headline with italic accent word
> - Countdown timer (default 90 days from load, configurable)
> - Notify form: email + first name, last name, area of interest (dropdown), phone, message — simulated submit only, nothing sent, shows success message on completion
>
> **Maintenance page**
> - Short downtime message
> - Estimated return time (2 hours from load by default)
>
> **Footer** — dark background, three sections:
> - Brand block: logo + company name + short tagline
> - Contact row with inline SVG icons: phone, email, address
> - Legal grid: Company Name, Company Number, VAT, Registered Address
> - Copyright bar bottom
>
> **Technical requirements**
> - Single `index.html`, zero external files — embed all images as base64
> - Google Fonts — elegant serif + clean sans-serif pairing (no Inter, no Roboto, no Arial)
> - Fully responsive down to mobile
> - Staggered entrance animations (fade + slide up)
> - CSS variables for all colours
> - Logo used in both header and footer, CSS-filtered to suit each background
>
